# 32. Reverse Linked List

Given the beginning of a singly linked list `head`, reverse the list, and return the new beginning of the list.

# Solution

https://www.youtube.com/watch?v=G0_I-ZF0S38

## Solution 1: Iterative Solution

```py
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next

class Solution:
    def reverseList(self, head: Optional[ListNode]) -> Optional[ListNode]:
        previous = None # Node before current
        current = head

        while current:
            temp = current.next # Store next node before breaking link
            current.next = previous # Link current node to previous node
            previous = current # Set prevoius node to current
            current = temp # Set current to the original next node

        return previous # When done, current is None, and previous is the node right before, which is the head of the reversed list
```

For a linked list of size $N$:
- Time complexity: $O(N)$
- Space complexity: $O(1)$

## Solution 2: Recursive Solution

```py
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next

class Solution:
    def reverseList(self, head: Optional[ListNode]) -> Optional[ListNode]:
        if not head: # Base case: End of the list
            return head

        new_head = head # Temporarily store current node
        if head.next: # If we are not at the end of the list
            new_head = self.reverseList(head.next) # Reverse the list after the current head, and return the new head of the newly reversed list
            head.next.next = head # Reverse the link of the node right after the head

        head.next = None # Currently we have a cycle, break the original link

        return new_head # Return the head of the newly reversed list
```

- Time complexity: $O(N)$
- Space complexity: $O(N)$
