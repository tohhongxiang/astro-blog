# 36. Remove Nth Node From End Of List

https://leetcode.com/problems/remove-nth-node-from-end-of-list/description/

You are given the beginning of a linked list `head`, and an integer `n`.

Remove the `nth` node from the end of the list and return the beginning of the list.

# Solution

## Solution 1: Iteration (Two Pass)

```py
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next

class Solution:
    def removeNthFromEnd(self, head: Optional[ListNode], n: int) -> Optional[ListNode]:
        # Calculate the size of the list
        size = 0
        curr = head
        while curr:
            curr = curr.next
            size += 1

        remove_index = size - n # Find the index of the node to remove
        if remove_index == 0: # If we have to remove the head
            return head.next # Return the node right after the head

        # Go through the list
        curr = head
        for i in range(size):
            if i + 1 == remove_index: # Remove the node at remove_index
                curr.next = curr.next.next
                break
            
            curr = curr.next

        return head # Then return the head
```

- Time complexity: $O(N)$
- Space complexity: $O(1)$

## Solution 2: Iteration (One Pass)

1. We create a `dummy` `ListNode` before the beginning of the array. 
2. We create a `left` and `right` pointer. Initialize `left = dummy` and `right` to be at `head`
3. We move `right` forward `n` steps.
4. Then we move both `left` and `right` forward until `right` is `None`. 
5. At this point, `left` is now right behind the node to remove. `left.next` is the node to delete, and we can remove it from the list by doing `left.next = left.next.next`
6. We return `dummy.next` as the updated head

```py
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next

class Solution:
    def removeNthFromEnd(self, head: Optional[ListNode], n: int) -> Optional[ListNode]:
        dummy = ListNode(0, head) # Create dummy node before head (this makes removing the front of the list easier)
        left = dummy
        right = head

        # Move right forward by n steps
        for _ in range(n):
            right = right.next

        # Move both pointers to the end
        while right:
            right = right.next
            left = left.next

        # left.next is the node we want to remove
        left.next = left.next.next
        return dummy.next # Return the original list
```

- Time complexity: $O(N)$
- Space complexity: $O(1)$

## Solution 3: Recursion

1. Recursively go to the end of the list.
2. As we unwind the recursion, decrement `n` every time we return.
3. When `n == 0`, this is the node to delete, return `curr.next`. Else, return `curr` and continue rebuilding the list

```py
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next

class Solution:
    def removeNthFromEnd(self, head: Optional[ListNode], n: int) -> Optional[ListNode]:
        def recurse(head):
            if not head: # We have reached the end of the list
                return None

            head.next = recurse(head.next) # Recursively process the rest of the list
            
            nonlocal n
            n -= 1 # Decrement n by 1
            if n == 0: # If we are at the node to remove
                return head.next # Return only the next node, so the previous recursion call will connect only with the node after the current node

            return head # If not, return the current node

        return recurse(head)
```

- Time complexity: $O(N)$
- Space complexity: $O(N)$
