# 26. Search a 2D Matrix

https://leetcode.com/problems/search-a-2d-matrix/description/

You are given an `m x n` 2-D integer array `matrix` and an integer `target`.

- Each row in `matrix` is sorted in non-decreasing order.
- The first integer of every row is greater than the last integer of the previous row.

Return `true` if target exists within matrix or `false` otherwise.

Can you write a solution that runs in $O(\log(m  n))$ time?

# Solution

## Solution 1: Binary Search Twice

Key insight: Do binary search across the rows first:
- If `first_row_element < target < last_row_element`, we found the correct row
- If `first_row_element > target`: Reduce to first half
- Else: Reduce to second half

Once the row is found, binary search again within that row. 

```py
class Solution:
    def searchMatrix(self, matrix: List[List[int]], target: int) -> bool:
        row_start = 0
        row_end = len(matrix) - 1
        while row_start <= row_end:
            row_middle = (row_start + row_end) // 2

            if matrix[row_middle][0] <= target <= matrix[row_middle][-1]:
                break
            elif matrix[row_middle][0] > target:
                row_end = row_middle - 1
            else:
                row_start = row_middle + 1

        if row_start > row_end:
            return False

        row_middle = (row_start + row_end) // 2
        column_start = 0
        column_end = len(matrix[row_middle]) - 1
        while column_start <= column_end:
            column_middle = (column_start + column_end) // 2

            if matrix[row_middle][column_middle] == target:
                return True
            elif matrix[row_middle][column_middle] < target:
                column_start = column_middle + 1
            else:
                column_end = column_middle - 1

        return False
```

- Time complexity: $O(\log M + \log N) = O(\log(MN))$
- Space complexity: $O(1)$

## Solution 2: Binary Search (One Pass)

Key insight: We pretend the 2D matrix is just a flattened array. We can get the corresponding `row` and `column` in the matrix using:
- `row = index // NUMBER_OF_COLUMNS`
- `column = index % NUMBER_OF_COLUMNS`

```py
class Solution:
    def searchMatrix(self, matrix: List[List[int]], target: int) -> bool:
        start = 0
        end = len(matrix) * len(matrix[0]) - 1

        while start <= end:
            middle = (start + end) // 2
            row = middle // len(matrix[0])
            column = middle % len(matrix[0])

            if matrix[row][column] == target:
                return True
            elif matrix[row][column] > target:
                end = middle - 1
            else:
                start = middle + 1

        return False
```

- Time complexity: $O(\log(MN))$
- Space complexity: $O(1)$
