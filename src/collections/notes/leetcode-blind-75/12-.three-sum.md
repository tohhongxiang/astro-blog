# 12. Three Sum

https://leetcode.com/problems/3sum/description/

Given an integer array nums, return all the triplets `[nums[i], nums[j], nums[k]]` such that `i != j, i != k, and j != k`, and `nums[i] + nums[j] + nums[k] == 0`.

Notice that the solution set must not contain duplicate triplets.

# Solution

```py
class Solution:
    def threeSum(self, nums: List[int]) -> List[List[int]]:
        nums.sort()
        result = []

        for i in range(len(nums)):
            current_num = nums[i]
            if i > 0 and current_num == nums[i - 1]: # Skip identical numbers (except the first one)
                continue

            left = i + 1
            right = len(nums) - 1

            while left < right:
                current_sum = current_num + nums[left] + nums[right]

                if current_sum < 0: # Sum too small, increase left pointer
                    left += 1
                elif current_sum > 0: # Sum too big, decrease right pointer
                    right -= 1
                else:
                    result.append([current_num, nums[left], nums[right]]) # One of the valid results
                    left += 1 # Shrink left and right pointer
                    right -= 1
                    while left < right and nums[left - 1] == nums[left]: # Make sure we are on a different number
                        left += 1

        return result
```

For a list of $N$ `nums`:
- Time complexity: $O(N^2)$
- Space complexity: $O(1)$ or $O(N)$ depending on the sorting algorithm
