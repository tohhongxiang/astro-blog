# 31. Median of Two Sorted Arrays

You are given two integer arrays `nums1` and `nums2` of size `m` and `n` respectively, where each is sorted in ascending order. Return the median value among all elements of the two arrays.

Your solution must run in $O(\log(m+n))$ time.

# Solution 

https://www.youtube.com/watch?v=q6IEA26hvXc

```py
class Solution:
    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:
        A = nums1 # shorter array
        B = nums2 # longer array

        if len(A) > len(B): # swap if nums2 was the longer array
            A, B = B, A

        total = len(A) + len(B) # get the total number of elements
        half = total // 2 # get the number of elements we want in the left partition

        # Binary search on the shorter array
        left = 0
        right = len(A) - 1
        while True:
            i = (left + right) // 2 # Get the left partition on the shorter array
            j = half - i - 2 # Get the left partition on the longer array

            a_left = A[i] if i >= 0 else float('-inf') # Largest element of the left partition in the shorter array
            a_right = A[i + 1] if i + 1 < len(A) else float('inf') # Smallest element of the right partition in the shorter array
            b_left = B[j] if j >= 0 else float('-inf') # Largest element of the left partition in the longer array
            b_right = B[j + 1] if j + 1 < len(B) else float('inf') # Smallest element of the right partition in the longer array

            if a_left <= b_right and b_left <= a_right: # Our left partition contains all the elements that should be on the left when merged
                if total % 2:
                    return min(a_right, b_right)
                else:
                    return (max(a_left, b_left) + min(a_right, b_right)) / 2
            elif b_left > a_right: # Our left partition is too small, smallest element in the left partition of B > largest element in the right partition of A
                left = i + 1
            else: # Our left partition is too big
                right = i - 1

        return -1 # Should never be reached
```

- Time complexity: $O(\log (\min(M, N)))$
- Space complexity: $O(1)$
